# Counsel

  > Legal matter command for the SZL Holdings platform.

  [![CI (platform monorepo)](https://github.com/szl-holdings/szl-holdings-platform/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/szl-holdings/szl-holdings-platform/actions/workflows/ci.yml)
  [![License](https://img.shields.io/badge/license-Proprietary-red?style=flat-square)](./LICENSE)

  Counsel provides matter tracking, obligation dependency mapping, document review, and exposure management with policy-gated AI workflows and proof-chain delivery.

  ## Capabilities

  - Matter tracking and lifecycle management
  - Obligation dependency mapping and deadline monitoring
  - AI-assisted document review with human approval gates
  - Legal exposure management and risk scoring
  - Proof-chain delivery for audit compliance

  ## Status

  **Alpha** — matter tracking functional. This is a product module within the [SZL Holdings platform monorepo](https://github.com/szl-holdings/szl-holdings-platform).

  ## Tech Stack

  TypeScript · React · Vite · Express · PostgreSQL · Drizzle ORM
